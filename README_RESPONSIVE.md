# Responsive update

The shared `css/responsive.css` file is loaded last on the active pages and provides layouts for:

- wide desktop screens
- notebooks and tablets
- mobile screens down to 320–380px
- short landscape screens
- reduced-motion accessibility preferences

Primary test sizes: 1920×1080, 1440×900, 1024×768, 768×1024, 390×844, and 360×800.
